# 반복문 배우기 - for

- 반복문은 특정 코드를 반복하는 흐름을 제어함

```
for(초기값; 조건식; 어떤 간격인지){
    실행할 문장
}
```

- 첫번째 지시어 for로 인해 반복문이 시작됨

```
for(let i =0; i <10; i++){
    colsole.log(i+번째);
}
```

- 10이 되면 종료
